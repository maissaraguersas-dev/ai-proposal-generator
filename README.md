## ✨ Features

- **🎯 One-Click Generation** — Paste a job description, get a tailored proposal instantly
- **🎨 Beautiful Dark UI** — Professional interface built with Tailwind CSS
- **🔒 100% Private** — Your API key stays in your browser; nothing is stored on servers
- **⚡ Streaming Ready** — Fast generation with Anthropic Claude API
- **📋 Copy & Download** — One-click copy or download as `.txt`
- **🎛️ Customizable** — Choose your role, experience, tone, and platform
- **📱 Responsive** — Works perfectly on desktop, tablet, and mobile

## 🔑 Setup

1. **Get your free Anthropic API key** at [console.anthropic.com](https://console.anthropic.com/)
2. **Paste it** in the API Key field
3. **Paste any job description** from Upwork, Malt, LinkedIn, or Fiverr
4. **Click Generate** and watch the magic happen

## 🎨 Customization

### Change the default role
Edit the `<select id="role">` element in `index.html`.

### Add more platforms
Add options to the `<select id="platform">` element.

### Modify the AI prompt
Find the `prompt` variable in the `<script>` tag and customize it to your style.

## 🌐 Deployment

### GitHub Pages (Free)
1. Fork this repo
2. Go to Settings → Pages → Source: Deploy from branch → main
3. Your site is live at `https://yourusername.github.io/ai-proposal-generator`

### Netlify (Free)
1. Drag and drop the folder to [Netlify Drop](https://app.netlify.com/drop)
2. Get an instant HTTPS URL

### Vercel (Free)
```bash
npm i -g vercel
vercel --prod
```
